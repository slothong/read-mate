# Read Mate

웹 페이지에서 텍스트를 선택하면 AI가 쉽게 설명해주는 Chrome 확장 프로그램입니다.

## 기능

- 텍스트 선택 시 AI 설명 팝오버 표시 (OpenAI API 스트리밍)
- 페이지 문맥을 활용한 정확한 설명
- 키보드 단축키 지원 (`Cmd+Shift+E` / `Ctrl+Shift+E`)
- 한국어 / 영어 / 자동 감지 응답 언어 설정

## 설치

1. 저장소 클론
2. Chrome → `chrome://extensions` → 개발자 모드 활성화
3. "압축해제된 확장 프로그램을 로드합니다" → 클론한 폴더 선택

## 설정

확장 프로그램 아이콘 클릭 후 OpenAI API Key, 모델, 응답 언어를 설정합니다.

## 사용

1. 웹 페이지에서 텍스트 드래그
2. 나타나는 `?` 버튼 클릭 (또는 단축키)
3. 팝오버에서 설명 확인
